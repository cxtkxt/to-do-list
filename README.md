## to-do list

Šī ir programma priekš ikdienas uzdevumu organizēšanai. Lietotājs var pievienot jaunus
uzdevumus, dzēst nevajadzīgos un atzīmēt izpildītos uzdevumus. 
Programma palīdz ērtāk sekot līdzi dažādām lietām kā piemēram; skolas darbiem, mājasdarbiem un citiem ikdienas plāniem.
Visi uzdevumi tiek saglabāti failā, tāpēc pēc programmas aizvēršanas tie nepazūd.


Funkcionālās prasības:
- Lietotājs var pievienot jaunus uzdevumus
- Lietotājs var dzēst uzdevumus
- Lietotājs var atzīmēt uzdevumus kā izpildītus
- Uzdevumi tiek attēloti sarakstā
- Programma saglabā uzdevumus failā
- Pēc programmas atkārtotas atvēršanas uzdevumi tiek ielādēti

Nefunkcionālās prasības:
- Programmai jābūt viegli saprotamai
- Programmai jābūt ar grafisko saskarni
- Programmai jādarbojas bez interneta pieslēguma
